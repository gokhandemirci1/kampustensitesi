# Logo Dosyası

Lütfen logo dosyanızı bu klasöre `logo.png` adıyla kaydedin.

Logo dosyası şu özelliklere sahip olmalıdır:
- Format: PNG (şeffaf arka plan önerilir)
- Boyut: Yaklaşık 200x50px veya orantılı
- Dosya adı: logo.png

Logo yüklenmezse, site otomatik olarak "Kampüsten" metnini gösterecektir.
